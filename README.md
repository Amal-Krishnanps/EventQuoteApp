# EventQuoteApp

EventQuote is a professional, offline-first Android application designed for event management professionals and service providers to create, manage, and share estimates and invoices on the go.

## 🚀 Key Features
- 🔌 **Fully Offline:** No internet connection required. All customer and estimate data is securely stored locally.
- 📄 **PDF Generation:** Create professional, custom-branded PDF quotations/invoices.
- 💬 **Easy Sharing:** Share PDFs directly via WhatsApp, Email, or other platforms.
- 🗃️ **Services Catalog:** Maintain a master list of services and rates for quick estimate building.
- 💾 **Backup & Restore:** Safely export and import your database backups.

## 🛠️ Tech Stack
- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material Design 3)
- **Database:** Room Database (SQLite)
- **PDF Generation:** iText PDF
- **Architecture:** MVVM (Model-View-ViewModel)
- **Build System:** Gradle Kotlin DSL (`.kts`)
